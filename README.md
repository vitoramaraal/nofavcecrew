# NoFvce Crew

Plataforma automotiva lifestyle em formato PWA, focada em comunidade,
identidade visual e experiencia premium underground.

## Status

MVP frontend em React/Vite com:

- landing page minimalista
- formulario de aplicacao
- painel admin com Supabase Auth
- area privada de membros
- login por codigo secreto individual
- garagem de membros aprovados
- chat interno persistente
- QR de verificacao de membro
- exportacao da carteirinha em PNG
- carteirinha digital
- manifest e service worker PWA
- integracao preparada com Supabase

Veja `docs/setup.md` para rodar o projeto e `docs/supabase-schema.sql` para
preparar o banco. Para limpar membros/candidaturas de teste no Supabase, use
`docs/cleanup-test-data.sql`.
