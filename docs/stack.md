# NoFvce Crew — Tech Stack

## Philosophy

The stack was chosen focusing on:
- simplicity
- scalability
- modern development
- performance
- visual quality
- low initial cost

The goal is to build a modern installable web platform (PWA)
with a premium user experience.

---

# Frontend

## React

React was chosen because:
- huge ecosystem
- component-based architecture
- modern frontend standard
- excellent community support
- scalable for future growth

---

## Vite

Vite will be used as the frontend build tool.

Reasons:
- extremely fast
- lightweight
- modern developer experience
- excellent React support

---

## Tailwind CSS

Tailwind CSS will be used for styling.

Reasons:
- rapid UI development
- excellent dark mode support
- minimalistic design workflow
- perfect for modern interfaces

---

# Progressive Web App (PWA)

## vite-plugin-pwa

The platform will use PWA technology to:
- allow installation on mobile
- create app-like experience
- improve performance
- support offline features in the future

---

# Deployment

## Vercel

Vercel will be used for deployment.

Reasons:
- simple setup
- free tier available
- optimized for React/Vite
- automatic deployments
- fast global CDN

---

# Backend (Future)

Initial backend options:
- Firebase
- Supabase

Future possibilities:
- AWS infrastructure
- custom backend services
- scalable APIs

---

# Design Principles

The platform should prioritize:
- dark mode
- smooth animations

- minimalism

- performance

The project will be built incrementally.

Priority order:
1. Identity
2. Experience
